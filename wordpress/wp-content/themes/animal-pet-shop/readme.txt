== Description ==

Animal Pet Shop is a clean and sophisticated theme for creating a pet salon, animal care center, pet shop, dog walkers, pultry and animal husbandary farms, pet grooming, and pet sitting services, pet and animal shelter homes, puppy daycare centers, etc. With an elegant design having interesting imagery regarding pet care, this free theme is created as per the latest WordPress standards and brings you the latest codes that are clean and secure. This theme comes along with a highly optimized design that will make a lightweight website and make the pages load quickly delivering faster page load time to give your users a great user experience. If you want to add more to the existing design, there are a lot of personalization options available. In order to make your website ready for an international audience, this theme brings you WPML and RTL compatibility making your websiteâ€™s content translatable into various languages. The bootstrap framework is used as a base for creating this elegant theme. For bringing you more organic traffic, its creators have made it SEO-friendly. Call to Action Button (CTA) will also take care of the conversion rates. You will find this theme interactive for the audience as well as easy to use fr novices and beginners.

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Animal Pet Shop in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel and customize to taste.
